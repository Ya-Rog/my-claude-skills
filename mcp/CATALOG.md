# MCP-серверы — каталог

MCP (Model Context Protocol) — это не скилы. Скил = инструкция как делать (SKILL.md).
MCP = подключение к внешней системе (GitHub, браузер, база данных, Google Drive).

Устанавливаются через `~/.claude/settings.json` → `mcpServers`, не через `/plugin install`.

---

## Установлены ✅

| MCP | Что делает | Как установлен | Нужен API-ключ? |
|---|---|---|---|
| `context7` | Получает актуальную документацию любой библиотеки прямо в сессии. Вместо устаревших знаний из обучения — реальная дока точной версии. Критично для Laravel 9, Vue 2 которые Claude часто путает с новыми версиями | Встроен в Claude Code | Нет |
| Playwright (browser) | Управляет браузером — открывает страницы, нажимает кнопки, заполняет формы, делает скриншоты. Нужен для тестирования UI и автоматизации браузера | `playwright@claude-plugins-official` | Нет |

## Кандидаты на установку

| MCP | Что делает | Установка | Брать? |
|---|---|---|---|
| `Firecrawl` | Парсит любой сайт в чистый Markdown который Claude нормально понимает. Для конкурентного анализа, сбора документации, мониторинга изменений на сайтах конкурентов | `npm install -g firecrawl-mcp` + API-ключ на firecrawl.dev | **Условно** — бесплатный тир ограничен, при объёмах платный. Брать если нужен скрейпинг |
| `Google Workspace (gws)` | Открывает доступ ко всему Google — Drive, Gmail, Calendar, Sheets через один сервер. Можно автоматизировать таблицы, читать почту, создавать события | `npm install -g @googleworkspace/cli` | **ДА если** активно работаешь с Google Docs/Sheets. Иначе не нужен |

## Как добавить новый MCP

1. Найти репозиторий / документацию MCP-сервера
2. Установить (npm/pip/docker — зависит от сервера)
3. Добавить в `~/.claude/settings.json`:
```json
"mcpServers": {
  "название": {
    "command": "npx",
    "args": ["-y", "пакет"],
    "env": {
      "API_KEY": "твой-ключ"
    }
  }
}
```
4. Добавить строку в таблицу выше
