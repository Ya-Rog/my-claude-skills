# Официальные скилы — не трогаем

Эти скилы установлены из официальных маркетплейсов Anthropic и **намеренно не переводятся и не переносятся в my-claude-skills**.

## Почему не трогаем

Два критерия — если оба выполнены, скил остаётся официальным:

1. **Содержит скрипты/код** — скил запускает Python/bash, у него есть `scripts/`, `references/`, `agents/` директории с реальным кодом. Перевод тут ничего не даст — важен код, не текст.
2. **Не нужно понимать содержимое чтобы пользоваться** — просто говоришь "сделай PDF из этого" и оно работает. Читать инструкции не нужно.

Дополнительный бонус: **автообновления только плюс**. Anthropic улучшает эти скилы — мы получаем улучшения бесплатно без потери ничего своего.

## Правило для новых скилов

Когда находишь новый скил через `skill-discover` — задай себе вопросы:

```
Скил содержит scripts/ или agents/ с кодом?
  ДА → оставь официальным, не переноси
  НЕТ → это инструкции/паттерны → переводи и переноси в my-claude-skills

Мне нужно читать содержимое чтобы направлять работу?
  ДА → перевести полезно → переноси
  НЕТ → оставь официальным
```
